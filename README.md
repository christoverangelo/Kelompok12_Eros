# C Calculator Project

## 📌 Description
This project is a simple calculator program written in C.  
It performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

The program uses functions for each operation and provides a menu-based interface for user input.

---

## ⚙️ How to Compile and Run

```bash
gcc calculator.c -o calculator
./calculator
